This plugin tracks your live search keyword, so that you have the ability to view what your clients are searching for and have clear view about the results displayed there.

== Description ==

* tracks live search via Ajaxy live search plugin

* tracks wordpress default search form
 
== Installation ==

1. Upload `ajaxy-search-keyword` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
